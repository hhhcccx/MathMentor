# MathMentor-MiMo (数学导师 AI 助手)

本项目是一款基于 **Xiaomi MiMo-V2.5-Pro** 开发的个性化数学辅助教学系统。旨在利用模型超长上下文（1M Token）和强大的推理能力，为学生提供精准的个性化辅导。

### 核心功能
* **错题长效追踪：** 利用长上下文能力，分析学生近一年的错题模式。
* **多模态作业批改：** 结合 MiMo-V2.5 原生多模态能力，识别手写数学公式。
* **智能讲义生成：** 自动根据教学进度生成针对性的填空题与实际应用题。

### 技术栈
* Model: Xiaomi MiMo-V2.5-Pro (via API)
* Frontend: Streamlit
* Language: Python 3.9+
